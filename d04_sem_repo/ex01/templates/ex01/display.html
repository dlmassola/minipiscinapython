Descreva o processo que causa a exibição de uma web estática página a partir de um template simples passando por uma view, desde a requisição até o responder.

  • o processo começa com o usuário digitando o endereço da página no navegador
  • o navegador envia uma requisição para o servidor
  • o servidor recebe a requisição e envia para o Django
  • o Django recebe a requisição e verifica se existe uma url que corresponde a requisição, se existir ele envia para a view
  • a view processa a requisição e envia para o template
  • o template processa a requisição e envia para o navegador
  • o navegador recebe a resposta e exibe para o usuário
